Detailed Description

Edge defining the cost function for penalzing a specified turning direction, in particular left resp. right turns.

The edge depends on two consecutive vertices and penalizes a given rotation direction based on the weight and dir ( ) dir should be +1 to prefer left rotations and -1 to prefer right rotations
weight can be set using setInformation().
